固件怎么画表情？

  • fillRect — 普通眼、sleepy 半眼、scan 窄眼
  • drawLine × N — squish 的 > < chevron
  • 5×5 位图 — 爱心眼 HEART5[]
  • 两段斜线 — 开心弧 drawHappyArc
  • animPhase + tick — Monitor 非阻塞动画

眼睛位置由 eyeLX/RX(ox)、eyeY() 根据 EYE_W/H/GAP/OY 算出，左右对称。
